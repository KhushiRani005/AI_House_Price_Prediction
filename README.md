# AI_House_Price_Prediction
AI-Powered House Price Prediction and Real Estate Analytics System  An end-to-end Machine Learning project that predicts residential property prices using the Ames Housing Dataset. The project combines data preprocessing, exploratory data analysis (EDA), feature engineering, multiple regression models, explainable AI.
Key Features
Data Cleaning and Missing Value Handling
Exploratory Data Analysis (EDA)
Feature Engineering (House Age, Total Area, Total Bathrooms)
Multiple Machine Learning Models
Linear Regression
Ridge Regression
Random Forest Regression
XGBoost Regression
Model Performance Comparison using R² Score, RMSE, and MSE
Explainable AI using Feature Importance Analysis
Interactive Streamlit Dashboard
Real-Time House Price Prediction Interface
Data Visualization using Plotly and Matplotlib

Pandas
NumPy
Scikit-learn
XGBoost
